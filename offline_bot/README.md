# PocketAgent Offline (Orange Pi Zero 3)

This repo is a fork/evolution of PocketAgent with Whisplay display support, updated to run **fully offline** on **Orange Pi Zero 3**.

## Goals
- Offline STT via **whisper.cpp**
- Offline “thinking brain” via **llama.cpp** (GGUF)
- Offline TTS via **Piper**
- Deterministic reminders engine + SQLite persistence
- "Two brain" router (reflex commands vs LLM JSON)
- Memory: ability to store facts/notes you tell it (“remember that…”) and retrieve later
- Whisplay LCD UI: status pill + face + subtitle + sleep after inactivity
